# ALU Machine Learning - Math Visualization

This project contains Python scripts for data visualization and mathematical plotting exercises, part of the ALU Machine Learning curriculum.

## Project Structure

```
math/
├── plotting/              # Visualization scripts
│   ├── 0-line.py         # Line plot examples
│   ├── 1-scatter.py      # Scatter plot examples
│   ├── 2-change_scale.py # Scale transformation examples
│   ├── 3-two.py          # Multiple plots/subplots
│   ├── 4-frequency.py    # Frequency distributions
│   ├── 5-all_in_one.py   # Comprehensive examples
│   ├── 6-bars.py         # Bar chart examples
│   └── Vagrant/
```

## Plotting Scripts

- **0-line.py** - Basic line plotting examples
- **1-scatter.py** - Scatter plot demonstrations
- **2-change_scale.py** - Exploring different scaling techniques
- **3-two.py** - Creating multiple plots or subplots
- **4-frequency.py** - Frequency and histogram visualizations
- **5-all_in_one.py** - Comprehensive examples combining multiple plot types
- **6-bars.py** - Bar chart and categorical data visualization

## Getting Started

1. Ensure you have Python 3 installed
2. Install required dependencies:
   ```bash
   pip install matplotlib numpy
   ```

3. Run any plotting script:
   ```bash
   python plotting/0-line.py
   ```

## Requirements

- Python 3.x
- matplotlib
- numpy

## Notes

This project uses Vagrant for environment management. See the `plotting/Vagrant/` directory for configuration details.
